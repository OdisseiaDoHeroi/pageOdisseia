Odisseia — A jornada do herói

Uma Jornada Épica em 2D

Baseado na clássica Odisseia de Homero, este jogo 2D foi desenvolvido no Godot com C# e Dialogic, trazendo a mitologia grega para uma experiência interativa e envolvente.

Conheça o Herói

Sobre o Projeto

Este jogo foi criado como Trabalho de Conclusão de Curso (TCC), unindo programação, narrativa e arte. O objetivo é adaptar a jornada de Odisseu para um universo jogável, usando a engine Godot, C# para a lógica e Dialogic para o sistema de diálogos interativos.

O jogador enfrenta monstros mitológicos, toma decisões e descobre os desafios da volta para Ítaca.

Personagem Principal

Odisseu

O rei de Ítaca, conhecido por sua astúcia e coragem. Após a Guerra de Troia, enfrenta inúmeros desafios para retornar ao lar e reencontrar sua família.
